Introduction

Chapter Overview

Chapter 2: Frequency Distributions and Graphs

After collecting data, the analyst’s next challenge is to organise and present it in a way that reveals meaningful patterns. This chapter introduces two fundamental tools for that task:

  • Frequency distributions — structured tables that summarise how often each value (or range of values) occurs
  • Statistical graphs — visual displays that make patterns, trends, and comparisons immediately apparent

Why Organise Data?

Raw data — a collection of numbers or labels as gathered — is difficult to interpret at a glance. Organising data into distributions and graphs allows researchers to:

  • Identify the centre and spread of a data set
  • Spot outliers or unusual observations
  • Communicate findings efficiently to a broad audience

§2-1 Organising Data

Raw Data and the Need for Organisation

When data are first collected they appear in raw (unsorted) form. Consider 50 state record-high temperatures — a list of 50 numbers tells us very little until we organise them.

Definition: Frequency Distribution

A frequency distribution is a table that organises data values into classes (categories or intervals) and records the frequency — the count — of observations falling in each class.

  • Categorical frequency distribution — used when data consist of named categories
  • Grouped frequency distribution — used when data are numerical and span a wide range; values are grouped into equal-width intervals
  • Ungrouped frequency distribution — used when the range of values is small; each distinct value forms its own class

Categorical Frequency Distribution

Twenty-five army inductees were blood-typed with the following results:

Grouped Frequency Distribution — Key Terminology

Definitions: Class, Limits, Boundaries, Width, Midpoint

Term Meaning
Class An interval used to group data
Class limits Smallest (lower) and largest (upper) values that belong to a class
Class boundaries Values halfway between consecutive class limits — eliminate gaps
Class width Difference between consecutive lower class limits (or boundaries)
Class midpoint Average of the lower and upper class boundaries: \(X_m = \dfrac{L + U}{2}\)

Choosing the number of classes: Use between 5 and 20 classes. As a starting rule, use \(k\) classes where \(2^k \geq n\).

Class width formula (always round up): \[\text{Width} = \frac{\text{Range}}{\text{Number of classes}} = \frac{\max - \min}{k}\]

§2-2 Histograms, Frequency Polygons, and Ogives

Three Key Graphs for Frequency Distributions

Definitions: Histogram, Frequency Polygon, Ogive

Histogram — a bar graph where each bar represents one class; the x-axis shows class boundaries and the y-axis shows frequency (or relative frequency). Bars are adjacent with no gaps.

Frequency polygon — a line graph connecting the midpoints of each class at a height equal to the class frequency. The line is anchored to zero at each end by adding an empty class on each side.

Ogive (cumulative frequency graph) — a line graph plotting cumulative frequency against the upper class boundary. Used to determine how many observations fall below a given value.

Relative frequency versions replace raw counts with proportions (\(f/n\)) or percentages on the y-axis, enabling fair comparisons across data sets of different sizes.

Histogram

Relative Frequency Graphs

Example 2-7 — Relative Frequency Histogram

A researcher recorded the number of miles run per week by a sample of joggers:

Miles Frequency
5.5 – 10.5 1
10.5 – 15.5 2
15.5 – 20.5 3
20.5 – 25.5 5
25.5 – 30.5 4

Construct a relative frequency histogram and a relative frequency ogive.

miles_mid  <- c(8, 13, 18, 23, 28)
miles_freq <- c(1, 2, 3, 5, 4)

rel_freq   <- miles_freq / n
cum_rel    <- cumsum(rel_freq)

  xmin = bounds[-length(bounds)],
  xmax = bounds[-1],
  y    = rel_freq
)
ggplot(df_hist, aes(xmin = xmin, xmax = xmax, ymin = 0, ymax = y)) +
  geom_rect(fill = "steelblue", color = "white") +
  scale_y_continuous(labels = scales::percent) +
  labs(title = "Relative Frequency Histogram",
       x = "Miles per Week", y = "Relative Frequency") +
  theme_classic(base_size = 12)
# Relative frequency ogive
df_ogive <- data.frame(x = bounds, y = c(0, cum_rel))
ggplot(df_ogive, aes(x, y)) +
  geom_line(color = "darkred", linewidth = 1) +
  geom_point(color = "darkred", size = 2) +
  scale_y_continuous(labels = scales::percent) +
  labs(title = "Relative Frequency Ogive",

  theme_classic(base_size = 12)

Distribution Shapes

Common Distribution Shapes

Frequency polygons and histograms reveal the shape of a distribution. The most important shapes are:

Shape Description
Bell-shaped (symmetric) Tapers off equally on both sides; data cluster near the centre
Uniform Roughly flat — all classes have similar frequencies
J-shaped Frequencies increase or decrease monotonically; resembles the letter J
Reverse-J Mirror image of J-shaped
Right-skewed (positive skew) Long tail extends to the right; most data are at the lower end
Left-skewed (negative skew) Long tail extends to the left; most data are at the upper end
Bimodal Two distinct peaks; may indicate two different subgroups
U-shaped High frequencies at both extremes, low in the middle

Misleading Graphs

Caution: Common Ways Graphs Mislead

Graphs can create false impressions if the reader is not careful. Watch for:

  1. Truncated y-axis — starting the vertical axis above zero exaggerates differences between bars or lines
  2. Inconsistent scale — unequal intervals on an axis distort the visual distance between data points
  3. Three-dimensional effects — 3-D charts make it difficult to judge the true height of bars or the size of pie slices
  4. Pictographs without area scaling — doubling a picture’s height quadruples its area, overstating a 2× difference

Rule of thumb: Always check whether the axis begins at zero and whether the scale is uniform before drawing conclusions from a graph.

Stem-and-Leaf Plot

Definition: Stem-and-Leaf Plot

A stem-and-leaf plot is a method for organising numerical data that retains every individual value. Each data value is split into two parts:

  • Stem — the leading digit(s), written to the left of a vertical line
  • Leaf — the trailing digit, written to the right

A stem-and-leaf plot serves as a quick histogram while preserving the exact data, making it easy to find the minimum, maximum, and median at a glance.

Summary

Chapter 2 Key Concepts

Frequency Distributions

  • A categorical distribution tabulates named categories; a grouped distribution uses equal-width intervals for numerical data; an ungrouped distribution lists each distinct value separately.
  • Class width \(= \lceil (\max - \min) / k \rceil\); class midpoint \(= (L + U) / 2\).
  • A cumulative frequency column shows how many observations fall at or below each class.

Graphs for Frequency Distributions

  • Histogram — adjacent bars, class boundaries on x-axis.
  • Frequency polygon — connects class midpoints; anchored to zero at both ends.
  • Ogive — cumulative frequency plotted against upper class boundaries.
  • Relative frequency versions use proportions rather than raw counts.

Other Graph Types

  • Bar graph / Pareto chart — categorical comparisons; Pareto sorts bars in descending order.
  • Time series — shows trends over time.
  • Pie graph — proportional slices; each slice’s angle \(= (f/n) \times 360°\).
  • Stem-and-leaf plot — retains exact values while revealing distribution shape.

Looking Ahead

In Chapter 3 we move from visual summaries to numerical summaries — measures of central tendency (mean, median, mode) and measures of variation (range, variance, standard deviation) — which allow us to describe a distribution precisely with a handful of numbers.
